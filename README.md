# ConsoleSay
 A Minecraft Plugin

![ScreenShot.jpg](https://i.loli.net/2020/02/06/YrG5UVQl3DBAFHw.jpg)

### Command

`c` Show usage and Reload configuration

`c [Args...]` Say word with format

(Console Only!)

### Configuration

```
Format: '[&3&l服务器后台&r]%WORD%.'
```

Use `%WORD% replace your words.
